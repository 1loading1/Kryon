# 人类思维缺陷之一些错误的相对性概念

前两篇文章分别讲了“存在”和“不存在”，这篇文章来讲一下我们人类很多相对性的概念中所存在的一些问题，比如有些相对性的概念其实是无法相对的，有些相对性的概念我们对它所指代的东西理解有误。

 

**“存在”和“不存在”是无法相对的**

 

看起来它们是一对相对的概念，但实际上它们无法相对。上篇文章中说了，“不存在”是不存在的，所以不存在不能被当作任何的主语和宾语。所以如果你把“无”理解为是什么都没有，不存在，那么“无中生有”这个说法就是错误的。**“存在”不是从“不存在”中来的，这个说法根本就不逻辑**。**“存在”没从哪来，它是无始的、不生的，也就是没有它诞生的时刻，也是无终的、不灭的，也就是没有它消失的时刻。“存在”就是一直存在，本来就有，本自具足的。**当然我这里说诞生消失的时刻，其实都是随顺线性思维的说法，四维线性思维在思考问题的时候总是要去问在哪儿？什么时候？而这些问题只在前四维中有效，如果我们硬是要用这几个有限维度的思维去理解存在的无限维度，那显然是无法理解的。那该怎么办呢？**憋住别问这些四维线性问题**是一种方法。把你当前的线性思维憋停了，说不定更深的洞见就浮现了，当你突破原有的思维障碍时，你会发现原来你问的那些问题根本就不是问题，因为这些本来就是错误的思维才会问出的错误问题，当你的思维正确了之后，这些问题自然就没了，而不是找到答案了。所以反过来说，如果你还憋不住要问何时，在哪，从什么时候开始这类线性问题时，那就是还没突破，继续好好憋住！

 

**我们对“主客”、“真幻”的理解很混乱**

 

主观和客观是一对相对性的概念，真实和幻觉、幻象这一对相对性的概念和主观和客观则是有联系的，因为我们一般会讲“客观真实”，也就是客观存在的东西才是真的，我们主观想象出来的都是幻象。这两对相对性概念所存在的问题是，我们对到底什么是主观的，什么是客观的，什么是真，什么是幻，都没搞清楚。

 

唯物主义对于“客观”的定义是“不随人们的主观意识转移的”，这个对于“客观”的定义是没问题的，但问题是，对什么东西才是客观的这个认识上有大问题。唯物主义一般认为物质是客观的，也就是物质是不随人的意志而转移的，然后又说人是有主观能动性的，可以改造物质世界。这话怎么听都不是很逻辑。而关键问题则是：**物质并不是客观的，意识是可以直接改变物质的，甚至是创造物质的，物质完全是能随意识而转移的，不是不可以。**

 

唯物主义认为这个世界的本质是物质，意识只是物质分子的运动不知怎的产生的现象，而这个只是现象的意识是不能直接改变物质的。可以用显示器来比作“物质”，而显示器画面上的某个人就是“意识”，这个人可以在显示器的画面里干任何事情，但他就是无法从显示器里面钻出来，然后把显示器（物质）砸掉。也就是唯物主义认为的：物质产生的意识现象是无法改变物质这个本质的。

 

但这个唯物的看法显然是错误的，我们可以看出这个观点的来源是来自于我们一直以来对于自然现象的观察和总结，也就是它是基于经验的。而人类一直以来的经验都是，我们改变这个世界的能力是很渺小的，所谓的客观自然界里面的事情有太多是我们人类无法改变的，而绝大多数人也都没有用意念弯曲勺子，或者是“凭空”显化物质的能力，所以我们自然就会认为物质是客观的，不随我们的意识所转移的。

 

但前文已经说过了，基于经验得出的结论往往是不靠谱的，不逻辑的。唯物的这个观点也是这样。首先人的意识是有直接改变，甚至是创造显化物质的能力的，历史上虽然绝大多数人都没有这个能力，但少数大师们还是有的，并一直在给我们做着展示。近代一位这样的大师就是塞巴巴，他就能似乎“凭空”显化出圣灰、水果面包等食品、手串等饰品之类的东西。而且他在集会的时候会经常这样做，也就是显化出一些食品给大家吃，显化出一些饰品送给大家。西方还有人去研究过他，给这个过程拍过录像。但不相信意识能显化物质的人就算亲眼见到了肯定也会说这是在造假，在变魔术。关于这个证据的问题我还写过这篇文章

而现代的科学家也已经开始发现意识其实也是实实在在的能量，不只是物质所产生的一个次级幻象，如果它不是实实在在的能量又怎么能够改变或显化物质？所以说“物质”和“意识”其实本质上都是能量，它们是一不是二。心、物是一元的，也就是同一种东西，不是两个对立的不同的东西，也不是一个只是另一个产生的幻象。我们所说的“物质”和“意识”这两个概念其实就是一个硬币一体的两面，是从不同的侧面认识同一个东西。

 

那如果这样说，或许有人就会问了，要是物质和意识本质是相同的，那岂不是意味着所有的物质都有意识？是这样的，万物皆有灵吗！从一个简单的原子，到复杂的人体，它们所具有的意识只是从简单到复杂，从低层到高层上的区别，并没有有和无上的区别。也就是说，一个最基本的物质粒子也是有基本的意识的，不是说最基本的粒子没有意识，然后意识不知怎的就从某个复杂的分子组合中出现了。这样的想法其实是不逻辑的，意识并不能无中生有。正确的观点是，物质、能量、信息、意识这些概念都是等价的，都是同义的，只是描述的侧面不同。既然能量是不生不灭的，那意识也是，也就是灵魂永恒。

 

**绝对客观的东西只有宇宙法则**

 

既然物质不是客观的，那什么东西才是客观的，不随人的意志所转移的？只有宇宙法则才是绝对客观的。说简单点就是，只有在背后支配万事万象运作的最最基本的规律才是绝对客观的。而万事万象不是，万物都是由能量构成的，而意识也是能量，这话说白了就是，万事万象都是我们的意识投射出来的，所以万事万象是主观的，不是客观的。

 

啥叫绝对客观的最基本规律呢，说最基本的意思是，我们的物理学定律其实还算不上最最基本的规律，因为克里昂说物理学规律和常数其实是可变的，每个星系的物理定律和常数都有点不一样。所以最基本的规律就只有宇宙法则，宇宙法则是绝对客观的，也是永恒不变的。

 

用比喻来说就是1+1=2，这是一个最基本的数学规律，它是绝对客观的是不是，就算你是佛祖，神，造物主，你说你要创造一个宇宙，在这个宇宙里1+1不等于二，圆周率也不等于3.1415926…，你说这可能吗？这个比喻的意思就是，最基本的规律是不可能变的，也不可能被违背，是绝对客观的，不管你是谁都得遵守。所以说客观规律、客观规律，**客观的就只有规律**。

 

**形而上者谓之道，形而下者谓之器**

 

其实上面所说的，也就是“形而上者谓之道，形而下者谓之器”这句话的意思，形而上和形而下是比主观可客观定义的更加清楚的相对性概念。形而下的“器”就是指有形有像的物质世界，形而上的“道”就是指无形无像，也就是抽象的法则、规律、本质、无限的尚未显化的能量。

 

那么请问，能量这个词，到底是形而上还是形而下呢？那这就要看你的能量到底指的是什么能量了。如果是指最本质的，无限的，不生不灭的，尚未显化成任何现象、处在平衡状态的能量，那这就指的是形而上。“何其自性本自清净，何其自性能生万法”这两句的意思就是在说这个能量本来是处在平衡清净的未显化状态，但它却有着显化万事万象的无限潜能。

 

如果你说的是电能，那这个能量就已经是一个已经显化出来的某个具体能量了，就属于形而下的概念。对于意识这个概念也是一样，我们都说灵魂是永恒的，所以灵魂这个词是形而上的。而我们一般所说的我这一生的小我意识，那它就是形而下的，因为它只是灵魂所有经历的一小部分，不是全体。

 

再把问题搅浑一点，其实形而上下也是一不是二，如果你把它当成两个东西来看，那又错了。这也可以用屏幕显像来比喻，屏幕是形而上，上面显的像是形而下。它们其实是一体的，像是在屏幕上显示的，不是在外面。但无论像怎么变化，屏幕都不会变。屏幕是永恒的，像是转瞬即逝的。当然这只是个比喻，比喻总有它能比喻的道理和一些不恰当的地方，这个比喻中不恰当的地方就在于，真正形而上的“屏幕”是无形无相的。

 

禅宗里面说对于什么是佛？这个问题有这样的回答：“说似一物即不中，随拈一物无不是”，这个话的意思用现在的话说就是在问什么是形而上的能量呢？由于一切都是由这个能量构成的，所以说“随拈一物无不是”，都是这个能量构成的。“说似一物即不中”的意思是：你不能说某个具体的东西是这个能量，就好比你用橡皮泥捏了个猫，你只能说这是个橡皮泥猫，这个猫是由橡皮泥构成的；不能说橡皮泥就是猫，或者猫就是橡皮泥，对不对？也就是上面所说的，你不能说某个具体的能量，比如电能，就是形而上的能量；也不能直接说显示器上的画面就是显示器。

 

**什么是真实**

 

我们对于什么是真实，什么是幻象的定义其实是基于感官的，并不严格也并不逻辑。因为世俗中一般把能看得见摸得着的东西当作是客观真实的，而把我们意识想象出来的东西当作是幻象。大家有没有看出来这样的定义完全是基于感官体验的，也就是我们把眼耳鼻舌身意这六识中前五识感知到的东西当作是“真的”，把只有第六识意识感知或想象到的东西当作是“幻象”。大家觉得这样的定义是不是有问题？

 

这里“只有意识感知或想象”的意思是，当我们在梦中，不管是白日梦还是夜梦，或是精神错乱的人看到幻象时，在这些情况下我们“看”到了东西，但前五识其实都没有参与其中。如果真的是通过眼睛看到的那就不是幻象了对不对，幻象的定义就是只有意识“看到”或想象出来的东西。佛法里对于我们在梦中仅存的第六识意识还有个专有名词，叫做“独影意识”。为啥我们在梦里梦到的画面总感觉不是那么真实呢，而且在梦中似乎也很少有触觉和嗅觉，那就是因为在一般梦境中前五识都休息了（清明梦又是另外一回事）。

 

那这样区分“真实”和“幻象”有什么问题呢？问题就在于，我们能看得见摸得着的客观物质其实也是我们的意识投影出来的啊，只是我们不知道而已。《盗梦空间》这个电影里有解释这个原理。所以所谓的“客观物质”和我们想象出来的“幻象”本质上并没有什么区别，都是意识的投影，换个词说就是都是信息。只是“真实”的东西中所含的能量大，信息多。而我们大部分人想象出来的东西能量可能还比较小，所以还不能立马就显化成“真实”的东西。而那些大师们可能意识能量更强大一些，所以他们能够直接把意识想象的东西马上显化出来。所以说想象的东西和现实的东西只是能量大小上有区别，并没有一个是“真”一个是“幻”这样本质的区别，其实它们都是“幻”。只不过你能看得见摸得着的东西是更加“真实”的幻象而已。

 

**如果这一切都是幻象，那么是不是一切就都没有意义？**

 

这是刚刚接触到物质也是意识投影的幻象这个概念的人往往会产生的误解，那就是认为既然一切都是幻象，那这一切的存在还有什么意义？活着还有什么意义？产生这个误解的原因是因为从一开始就没搞清楚真实的东西是什么，我们活着到底是为了什么？具体后文再说了，本文就写到这。
